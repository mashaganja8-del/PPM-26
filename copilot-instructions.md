## Project Overview

A fully functional piano game where players can:
- Play piano keys using the keyboard (A-Z keys)
- See visual feedback as keys are pressed
- Learn piano basics with an interactive interface

## Quick Start

1. Open index.html in a browser or use the "Open in Browser" task
2. Press keys A through Z to play different notes
3. Yellow highlighting shows which keys were pressed
